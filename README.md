peanutdb
===============================

version number: 0.1.0
author: Ahmed Noor

Overview
--------

A JSON based database for quick prototyping and playing.

Installation / Usage
--------------------

To install use pip:

    $ pip install peanutdb


Or clone the repo:

    $ git clone https://github.com/ahmednooor/peanutdb.git
    $ python setup.py install
